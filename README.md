## 🚀 Quraşdırılma və İşə Salınma (Installation)

Layihəni öz kompüterinizdə yoxlamaq üçün aşağıdakı addımları izləyin:

1. Bu repozitoriyanı klonlayın:

   ```bash

   git clone https://github.com[İstifadəçi_Adınız]/bmi_calculation.git

   ```

2. Layihə qovluğuna keçid edin:

   ```bash

   cd bmi_calculation

   ```

3. Lazım olan paketləri yükləyin:

   ```bash

   flutter pub get

   ```

4. Tətbiqi emulyatorda və ya real cihazda başladın:

   ```bash

   flutter run

   ```
